1、display:block(块级元素)： 会独占一行 可设其宽、高、margin、padding属性 可容纳内联元素与其他块元素
2、display：inline(行内元素)： 与其他元素在同一行 设置宽、高无效 padding-left/right，margin-left/right设置会产生边距效果 竖直方向的padding-top/bottom，margin-top/bottom不会产生边距效果
3、display：inline-block（行内块元素）： 既可设置宽、高，又默认不换行 可设置vertical-align属性 在字体大小不为零的情况下，会产生元素间的空隙，可在父元素设置 font-size：0 ，也可用letter-spacing设为负值 去除此空隙

inline 不支持宽高
inline 和 inline-block是以baseline对齐的， baseline并不是汉字的下端沿，而是英文字母x的下端沿
vertical-align: top

(1)容器的属性： flex-direction、 flex-wrap、 flex-flow、 justify-content、 align-items、 align-content
(2)项目的属性： order flex-grow flex-shrink flex-basis flex align-self

项目高度不设置时默认是容器的高度
设置flex-grow自适应宽度
flex-basis设置具体的宽度
当所有子元素的宽度flex-shrink都设置为0时， 容器的最小宽度的所有子元素设置的宽度之和。
项目设置了高度
容器设置align-items: center;垂直居中
容器设置align-items后项目的height不再是默认的容器的高度了
容器设置flex-direction: column
容器设置align-items后项目的height不再是默认的容器的高度了
容器设置flex-direction: column-reverse
容器设置align-items后项目的height不再是默认的容器的高度了

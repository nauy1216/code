(1)相对于父元素宽度的，（如果自身已经是最外层元素，则相对于视口）[max/min-]width、left、right、padding（包括padding-top,padding-bottom）、margin(包括margin-top,margin-bottom) 等；
(2)相对于父元素高度的：（如果自身已经是最外层元素，则相对于视口） [max/min-]height、top、bottom 等；
(3)其中，关于height百分比定位的一个冷知识是：若某元素的父元素没有确定高度，则无法有效使用height=XX%的样式。除非该元素已经是最外层元素，此时是相对于视口。
(4)相对于继承字号的：font-size 等；
(5)相对于自身字号的： line-height 等；
(6)相对于自身宽高的：border-radius、background-size、transform: translate()、transform-origin、zoom、clip-path 等；
(7)特殊算法的：background-position（方向长度 / 该方向除背景图之外部分总长度 * 100）、filter 系列函数等；
(8)如果自身设置 position: absolute，相对于离它最近的那个 position 不为 static 的祖先元素，如果没有这样的元素，则相对于视口。 如果 position: fixed，“父元素”指视口。
1、margin使用百分比是相对于父级的width的, 包括margin-top,margin-bottom,margin-left,margin-right。
1、padding使用百分比也是相对于父级的width的。
